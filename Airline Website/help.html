Help

Cursor Control Help

  • Mouth Controls: Open mouth to activate the controller. Again open mouth to deactivate the controller.
  • Nose Controls: Use nose to move the cursor in up/down/left/right directions.
  • Squint Eyes: Squint eyes to enable/disable scrolling mode.
  • Left Eye Wink: Wink left eye to perform left click operation.
  • Right Eye Wink: Wink right eye to perform right click operation.

Checkin Portal Help

  • First Page: Select the airline by clicking on respective airline logos.
  • Second Page: Enter PNR for self checkin using the virtual keyboard. Virtual keyboard is automatically enabled once you click the input PNR field.
  • Third Page: Checkin successful and boarding pass print option. Print option prints the boarding pass and returns you to the home page.
